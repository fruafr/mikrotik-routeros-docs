## Parameters

Invoke with : `/ip [parameter]`

The following command parameters are accepted:

| **Parameter** | **Shortcut** | **Official Description** | **Note** |
|---|---|---|---|
| **export** | e | Print or save an export script that can be used to restore configuration | |
