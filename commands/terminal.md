## Parameters

Invoke with : `/task [parameter]` or `/ta [parameter]`

The following command parameters are accepted:

| **Parameter** | **Shortcut** | **Official Description** | **Note** |
|---|---|---|---|
| **cuu** | c | move cursor up | |
| **el** | e | erase line | |
| **inkey** | i | read key | |
| **style** | s | set output text style | |
